# Reporducable Data Analysis Pipeline (Rando Lab, Stanford)

> BASH/R workflow-managment for cluster-parallelized analysis of large-scale sequencing data.
> Bulk & single-cell, RNA-seq, ATAC-seq, BS-seq.

Scripts and conventions for running NGS analysis _reproducibly_ at scale on an HPC cluster.
Standardized project layer, parameterized pipeline steps, parallel job submission.

Developed for internal lab use; paths and data are environment-specific.
